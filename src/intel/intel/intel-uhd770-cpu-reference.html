Intel UHD Graphics 770 CPU Reference

Intel UHD Graphics 770 is the 32-EU Xe-architecture integrated GPU found in selected Alder Lake, Raptor Lake, Raptor Lake Refresh, and Bartlett Lake processors. For T1-T6 shader artifacts, treat it as Intel integrated Xe-LP processor graphics, not Arc A770 discrete graphics.

Avoid These Traps

These names are close enough to fool a shopping cart or a parts scraper.

No F or KF modelsExamples: 12600KF, 12700F, 13700KF, 14900F. Intel marks these as having no processor graphics.
UHD 730 is not UHD 770Examples: i5-12400, i5-13400, i3-14100. These are useful CPUs, but the iGPU is a smaller target.
Look for UHD 770 explicitlyUse the product name plus ARK SKU ID when buying, especially for OEM and embedded stock.
